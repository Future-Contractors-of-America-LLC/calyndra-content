# Module 2: Core Words — More, All Done, Stop

**Time:** 10 minutes daily for 7 days  
**Goal:** Build a reliable core trio for requests and transitions

## Why these three?
- **More** = requesting continuation (snack, swing, story)
- **All done** = predictable endings (reduces transition stress)
- **Stop** = assent and safety (must be honored)

## Daily routine map

| Routine | Symbol to model |
|---------|-----------------|
| Snack | more, all done |
| Playground | more, all done |
| Bath | stop (if they need pause), all done |
| Rough play | stop (caregiver models if needed) |

## Rules
1. Never take the device when they tap **stop**.
2. Pair **all done** with a consistent sensory cue (same song, same gesture).
3. Limit to these three symbols—no new words this week.

## Practice script (caregiver reads)
> "More crackers?" [model **more**]  
> "All done with crackers." [model **all done**]  
> "Stop—I'll wait." [if they push away, model **stop** and pause]
