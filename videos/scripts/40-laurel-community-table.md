# Laurel's Community Table (adult, ~45 min, 450 scenes)

**Series:** Caly and Friends  
**Band:** Adult (Caly Bloom, 18+)  
**Friend:** Laurel the cardinal  
**Mentor:** Elder Oak  
**Christian lesson:** A shared table turns strangers into family. (Acts 2:46 - broke bread in their homes with glad hearts).

## Story arc

| Act | Scenes | Beats |
|-----|--------|-------|
| Empty chairs | 1-60 | Laurel sets long table, hopes for guests |
| Invitations | 61-120 | Personal invites to lonely neighbors, Caly Bloom |
| Prep together | 121-210 | Guests arrive early to cook, stories flow |
| Meal shared | 211-300 | Pass dishes, inclusive prayer, all voices |
| Hard stories | 301-390 | Safe space for grief and hope at table |
| Table legacy | 391-450 | Monthly plan, everyone signs up to host next |

| # | Visual | Dialogue |
