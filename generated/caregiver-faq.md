## 1. When should we start AAC?

Start when **any** communication gap causes stressónot when a clinician gives "permission." If a person already uses gestures, sounds, or expressions to be understood, they are already communicating. AAC adds **tools**, not prerequisites.

For toddlers, begin with **modeling** one core word (like **help** or **more**) during calm routines. There is no minimum age for access, but there is a maximum benefit to waiting out of fear. If you are unsure about access methods (eye gaze, switch, app), consult a speech-language pathologistóI can help you prepare questions for that visit.

---

## 2. Will AAC stop speech from developing?

Research and clinical practice generally show AAC **supports** language development; it does not replace speech for most users. Think of AAC as a bridgeónot a lock on spoken words. Many people use both speech and symbols over time.

My guidance: model symbols **alongside** natural speech. Never withhold AAC until someone " earns" it by speaking. That creates unnecessary pressure and can harm trust.

---

## 3. How many symbols should we start with?

**Fewer is better at first.** For toddlers, 3ñ5 core symbols for two weeks is a strong start: **help**, **more**, **all done**, **stop**, **eat**. For school-age children, 10ñ15 familiar words organized by routine beats a crowded board.

Add one new symbol per week only after the current set is modeled consistently. If the board feels overwhelming, remove symbols temporarilyóthis is not failure.

---

## 4. What if they only tap randomly?

Random taps are still **data**. Note which symbols get tapped in which contexts. Offer the expected outcome when safe (tap **drink** ? water). If taps seem accidental, reduce board size and increase wait time after you model.

Avoid testing ("Show me **more**"). Instead, narrate: "You tapped **more**. Here is another bite." Random exploration often decreases when responses become predictable and low-pressure.

---

## 5. How do I know if a symbol is the "right" one?

There is no single correct symbol for all people. A "right" symbol is one the **user will look at, reach for, or select** in contextóand that caregivers will honor quickly.

Choose:
- High-frequency words in daily life
- Words that reduce frustration today
- Images or text the user recognizes (photos can beat generic clipart)

When in doubt, ask the user (any age) with two choices and respect **no**. An AAC specialist can help customizeóI support your preparation, not their clinical role.
